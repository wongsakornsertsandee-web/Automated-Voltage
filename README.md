# Automated-Voltage
24/7 Automated Power Quality Monitoring (Google Apps Script, JavaScript)

Responsibilities: Design and develop a 24/7 automated power quality monitoring and analysis system for industrial plants, with no hardware costs.

Automation & Backend: Install time-driven triggers (Cron jobs) to activate JavaScript scripts to check power quality thresholds hourly, along with real-time email alerts and automatic timestamping for voltage dips or surges.

Data Visualization: Integrate Google Sheets with Google to display real-time voltage status via gauge charts and incident history.

Results: Transform a manual power monitoring system to a fully automated one, enabling electrical engineers to remotely monitor system status and detect abnormalities immediately.
